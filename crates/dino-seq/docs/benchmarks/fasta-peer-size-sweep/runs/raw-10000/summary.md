# FASTA Rust Library Peer Benchmark

Generated by `scripts/benchmark-fasta-peers.sh`.

Input: deterministic synthetic FASTA, `10000` records, sequence length `150`.

The benchmark reads one in-memory raw FASTA byte buffer through each Rust parser. It is parser-library evidence only and does not compare indexing, filtering, or command-line workflow behavior. Rows containing `two-line` use the strict `>header`/`sequence` fast path for canonical two-line FASTA; use the non-two-line resident row for ordinary multiline FASTA. The `counter` row is the light-accounting scan path used for count/bases/checksum workloads.

# FASTA Rust Peer Size Sweep

Generated by `scripts/benchmark-fasta-peer-size-sweep.sh`.

This sweep compares Rust FASTA parser frameworks on deterministic synthetic two-line FASTA inputs of increasing size. Raw rows measure one resident byte buffer per size. Gzip rows measure one gzip-compressed resident byte buffer per size; rows explicitly name whether they use third-party flate2 streaming, third-party flate2 resident decode, third-party libdeflate resident decode through the libdeflater Rust wrapper, strict two-line record views, or the light-accounting two-line counter path. This is parser-framework evidence, not command-line workflow evidence for indexing, filtering, BGZF, or a new dino_seq DEFLATE implementation.

# Rust Library Peer Benchmark

Generated by `scripts/benchmark-rust-peers.sh`.

Input: deterministic synthetic FASTQ, `100000` records, read length `150`.

The benchmark reads one in-memory raw FASTQ byte buffer through each Rust parser. It is parser-library evidence only: it does not compare gzip, BGZF, trimming, filtering, or command-line workflow behavior. The default consumer records shape/accounting work; set `DINO_SEQ_RUST_PEER_CONSUMER=full` to hash every sequence and quality byte.
